# UAV-Fleet-Command-CLI
![C Build & Check](https://github.com/ahmetcann66/UAV-Fleet-Command-CLI/actions/workflows/ci.yml/badge.svg)
"Multi-UAV Fleet Management &amp; Simulation System developed in C. Implements a CLI-based Ground Control Station (GCS) logic to command multiple drone models (TB2, AKINCI, AKSUNGUR). Features real-time telemetry tracking (Fuel/Altitude) and mission status reporting."
